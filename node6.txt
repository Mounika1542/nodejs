# â­ 1. What is a Module?

A **module** = a file or package that contains code you can reuse.

Node.js uses **CommonJS module system**, so we use:

* `require()` â†’ to import
* `module.exports` â†’ to export

# â­ 2. Types of Modules in Node.js

## âœ” **1. Core (Built-in) Modules**

Examples:

* `fs` â†’ File system
* `http` â†’ Create server
* `os` â†’ System info
* `path` â†’ Work with file paths

**Example:**


## âœ” **2. Local Modules**

These are files you create yourself.

Example:

**math.js**

```js

    return a + b;
}

```

**app.js**

```js

console.log(add(10, 20));
```
